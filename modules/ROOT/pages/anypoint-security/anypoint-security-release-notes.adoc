== November 17, 2018

=== New Features

This release includes the following new features:

* Anypoint Security tokenization service. The tokenization service enables you to substitute a sensitive data element with a non-sensitive equivalent to provide data protection. For information about how to implement tokenization, see xref:anypoint-security::tokenization.adoc[Anypoint Tokenization Service].
