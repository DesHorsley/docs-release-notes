The 2.1.0 release of Mule includes support for component interceptors, an upgrade to CXF 2.1.1 for web services, improvements to routers, JMX, transactions, and cleaner shutdown. Various improvements have been made to Mule's configuration schemas, more and more documentation has been updated and created for 2.x, and as always, many bug fixes and increased test coverage.
