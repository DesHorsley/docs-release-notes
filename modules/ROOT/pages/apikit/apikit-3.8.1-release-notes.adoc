This release implements fully compliant https://github.com/raml-org/raml-spec/blob/master/versions/raml-10/raml-10.md[RAML 1.0 validation]. https://github.com/raml-org/raml-spec/wiki/Breaking-Changes[The RAML.org web site] describes changes from RAML 0.8 to RAML 1.0.

== Compatibility

* Mule Runtime 3.8.x
* Anypoint Studio 6.0.x and 6.1.0
* Java Parser 1.0.0-beta-2
* Javascript Parser 0.2.32
* Console 3.0.6
