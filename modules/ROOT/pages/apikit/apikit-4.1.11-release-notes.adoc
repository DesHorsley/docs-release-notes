== What's New

There are no new features in this version. There are just bug fixes.

== Compatibility

* Mule Runtime 4.1.2 and later
* Anypoint Studio 7.2.0 and later
* Java Parser RAML 1.0 v1.0.20
* Java Parser RAML 0.8 v0.8.22
* API Console 4.2.0

== Fixed in the Release

* When scaffolding more than 1 API with same root name, it only scaffolds one.
* When updating an API and scaffolding it, it duplicates the whole application.
