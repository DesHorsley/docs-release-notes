Mule Maven Plugin 3.2.5 is mainly focused in fixing bugs when generating the application package.

== Hardware and Software Requirements

Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8
* Maven 3.3.3, 3.3.9, 3.5.0

== Compatibility

* Mule 4.x
* Anypoint Studio 7.x

== Fixed Issues

* MMP-444 - Dependencies with different classifier are considered equal
