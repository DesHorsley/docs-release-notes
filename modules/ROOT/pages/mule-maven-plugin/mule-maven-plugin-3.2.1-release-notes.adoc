Mule Maven Plugin 3.2.1 is mainly focused in fixing bugs.

== Hardware and Software Requirements

Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8
* Maven 3.3.3, 3.3.9, 3.5.0

== Compatibility

* Mule 4.x
* Anypoint Studio 7.x

== Fixed Issues

* MMP-373 - Find a way to use deploy:deploy alongside with mule:deploy
* MMP-416 - Redeploy fails when static ip configured on CH app
* MMP-430 - Support properties for Runtime Fabric deployment
