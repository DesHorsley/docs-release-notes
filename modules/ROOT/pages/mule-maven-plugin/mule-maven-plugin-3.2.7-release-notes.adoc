Mule Maven Plugin 3.2.7 is mainly focused in fixing bugs.

== Hardware and Software Requirements

Hardware and Software Requirements

* Microsoft Windows 8+
* Apple Mac OS X 10.10+
* Linux (tested on Ubuntu 15)
* Java 8
* Maven 3.3.3, 3.3.9, 3.5.0

== Compatibility

* Mule 4.x
* Anypoint Studio 7.x

== Fixed Issues

* MMP-462 - Mule Maven Plugin re-deployment issue when deploying the same app to two runtime fabrics
* MMP-452 - Provide a way to pass business group id instead of name
